# AppMergeFields

Display available merge fields for email templates in a grouped format. Merge fields are placeholders (format: `*|FIELD_NAME|*`) that get replaced with dynamic values at runtime.

## Purpose

This component provides a user-friendly interface for inserting merge fields in email templates. It is designed to be used within a dropdown context, such as in the `AppRichTextEditor` component.

## Features

- ✅ **Grouped display** with clear category separation
- ✅ **Insert action** for rich text editor integration
- ✅ **Multi-language support** with dynamic i18n keys
- ✅ **Example values** for better understanding
- ✅ **Accessible** with ARIA labels
- ✅ **Responsive design** and touch-friendly interface

## Common Use Cases

### Rich Text Editor Integration

Used within `AppRichTextEditor` as a dropdown to enable one-click insertion of merge fields at cursor position.
